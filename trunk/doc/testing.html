Testing


The test cases used can be found in the testSuite/ directory which contains the actual schemas and documents used for testing as well as plain text files containing the error messages that have been accepted as correct, which can be used for regression testing.

Serene 0.6.2 includes a command line tool that allows you to run the tests yourself either directly by using sereneTest.jar and the -jar option

$ java -jar path/sereneTest.jar path/testSuite/tests

or

$ java -jar path/sereneTestRNGSchematron.jar path/testSuiteRNGSchematron/tests

for the tests of RELAX NG embedding Schematron.

or by using the sereneTest.Driver class which contains the main method

$ java -cp path/sereneTest.jar sereneTest.Driver path/testSuite/tests

or

$ java -cp path/sereneTestRNGSchematron.jar sereneTestRNGSchematron.Driver path/testSuiteRNGSchematron/tests

for the tests of RELAX NG embedding Schematron.

You can see the usage instructions and the list of available options by invoking the tool without any arguments.
