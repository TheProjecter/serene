Serene 0.5


This is the first version of the Serene validation engine for RELAX NG, so it contains several important limitations, but in the same time it attempts to offer a validator that is functional enough to illustrate the approach to validation messages and ambiguity handling. It is primarily intended to be used as a library through the JAXP 1.3 Validation Framework API, but it can also work as a command line tool.
